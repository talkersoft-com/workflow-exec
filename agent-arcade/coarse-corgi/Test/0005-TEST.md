# Test 0005 — Dictation + recording-nav

## TC-001 — Dictation resolves on Go event, not on send
`⌘D` record → stop. Pass: UI enters `pending` and only reaches `confirmed` when `dictation-go`
emits `result` (and `error` → error state). It does NOT show success on dispatch.

## TC-002 — Within-agent nav keeps recording
While recording agent A, navigate within A. Pass: recording continues.

## TC-003 — send mode cross-agent commit
`recordingNavBehavior=send`, recording A, `⌘→` to B. Pass: dictation commits to **A**, toast
"Sent to A", navigation proceeds; when Go's `result`/`error` arrives it lands on **A** even though
B is now current.

## TC-004 — lock mode blocks
`recordingNavBehavior=lock`, recording A, attempt nav. Pass: navigation blocked, toast "Locked
while recording".

## TC-005 — Esc discards
Recording, press Esc. Pass: recording cancelled + discarded, toast "Recording cancelled"; no job
dispatched.
