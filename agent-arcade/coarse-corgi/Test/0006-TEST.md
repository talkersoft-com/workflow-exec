# Test 0006 — Arcade terminal surface

## TC-001 — Live peek
Open an agent's terminal view. Pass: the pane scrape renders and refreshes (matches reference).

## TC-002 — Sync mode (⌘F)
Enter sync, type. Pass: keystrokes forward raw to the pane; exit returns to the view.

## TC-003 — Workspace shell (⌘W)
`⌘W` from the terminal view. Pass: a real node-pty + xterm.js shell spawns (no `posix_spawnp`
failure — `spawn-helper` executable); `⌘A`/Esc behaves per reference.

## TC-004 — Macros select/text/flag
Run an `@`-macro with each arg type. Pass: select picks a value; text takes free input; flag
emits its token when ON / nothing when OFF; select/text stay shell-quoted.

## TC-005 — Optimistic reconciliation on non-zero exit
Force a `wezterm-bridge` command to exit non-zero. Pass: an **error state** is shown, not assumed
success.
