## Goal
Rewrite Studio as a React SPA (Zustand) that is **1:1** with the reference and reads/writes the
**same** `~/.hv/agent-arcade.yaml` via the existing main IPC.

## Steps
1. Set up the Studio React app under `studio/` (bundled by `build:js` → `studio/dist/**`).
   React/Zustand confined here — no React anywhere else.
2. Rebuild every Studio screen/flow to match the reference: agents (CRUD, avatar, color,

   `select` / `text` / `flag`.
3. All persistence goes through the existing main IPC against the unchanged YAML schema
   (agent/command normalization preserved). Do not alter the schema.
4. Verify against an identical config copied from the reference; `npm run build` green.

## Definition of done

the same YAML; build green. No React/Zustand leaks outside `studio/`.
