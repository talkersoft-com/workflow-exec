# Task 0000 — Initialize Workspace

## Goal
Verify the workspace is clean and put the deck on an execution branch for this orchestration.

## Steps
1. Run `hv_status deck: "cloud-manager"`
2. If all repos are clean and on the default branch → run `hv_init`
3. If all repos are clean and on a feature branch (e.g. a fresh bootstrap branch) → run `hv_next`
   (mints a new branch from origin/main across the deck)
4. If any repo is dirty → STOP and investigate. Exception: this orchestration's own scaffold
   folder showing as uncommitted/staged changes in workflow-exec is expected — it ships with the
   final hv_ship (see gloppy-snapper FIX-001 precedent).
5. Run `hv_status` again — all repos on the same new feature branch, otherwise clean
6. Record the generated branch name in `../deck.md` under "## Branch"

## Definition of done
All repos on the recorded execution branch; deck.md Branch field no longer says TBD.
