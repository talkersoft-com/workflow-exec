# Test 0003 — Build + deploy

## TC-001 — Dist rebuilt
Run:
```
ls cloud-manager-web/dist/index.html
```
Pass: file exists, modified within the last 10 minutes.
Fail: missing or stale.

## TC-002 — Web service active
Run:
```
sudo systemctl is-active cloud-manager-web
```
Pass: prints `active`.
Fail: anything else.

## TC-003 — New code present in deployed bundle
Run:
```
sudo grep -rl "Discard unsaved changes" /opt/cloud-manager-web/assets
```
Pass: at least one filename printed.
Fail: no output → new code not in deployed bundle.

## TC-004 — Page loads in browser
Use Playwright to navigate to `https://ubuntu-server.talkersoft.com/ansible/playbooks` and capture a snapshot.
Pass: snapshot contains the sub-nav and at least the `pg` playbook row.
Fail: page errors or is blank.
