# Task 0000 — Initialize Workspace

## Goal
Verify the workspace is clean and on `gloppy-snapper` across every repo in the cloud-manager deck.

## Steps
1. Run `hv_status deck: "cloud-manager"`
2. Confirm every repo is on branch `gloppy-snapper` and clean
3. If any repo is dirty → STOP. Investigate before proceeding.
   - Exception: dirty repo with a merged PR → `hv_stash` → `hv_next` → `hv_unstash`
4. If the deck is somehow not on a fresh feature branch: `hv_next deck: "cloud-manager"`, then update the Branch field in `../deck.md` with the generated name

## Definition of done
All 18 repos on `gloppy-snapper` (or the recorded replacement branch), all clean.
