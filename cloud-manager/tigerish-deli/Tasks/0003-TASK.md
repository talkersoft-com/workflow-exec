# Task 0003 — Verify, Deploy, Ship

## Goal
Prove both blueprints are live, then deploy and ship.

## Steps
1. Apply the migration to the live DB FIRST (command in init.md / PLAN.md).
2. `python3 .cicd/deploy-cloud-manager.py --target api`.
3. Run `cloud-manager-mcp/.cicd/import-playbooks.py` to seed `rabbitmq-*` into
   `vm.playbooks` (verify via `cloud_playbook_list`).
4. Assert `GET /api/v1/marketplace` lists `rabbitmq-jammy` AND `rabbitmq-noble`
   (or `cloud_*` marketplace tool equivalent).
5. Instantiate one blueprint; confirm the VM provisions, the playbook run succeeds
   (`cloud_run_get` / `cloud_run_logs`), and RabbitMQ is reachable on 5672/15672 with
   the password recorded in Vault.
6. Write `Results/RESULT.md` and `Retro/LESSONS.md` BEFORE shipping.
7. `hv_ship` (stage `exec`) — see deck.md.

## Definition of done
Both blueprints listed; one instantiation verified end-to-end; Results + Lessons
written; PR opened via hv_ship.
