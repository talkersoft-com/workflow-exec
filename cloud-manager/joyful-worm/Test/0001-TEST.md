# Test 0001 — Safe install-config

## TC-001 — Missing files are copied
Run `make install-config` targeting a temp directory with no existing files.
Pass: all files from `.hv/` appear at the destination.
Fail: any file missing.

## TC-002 — Existing files are not overwritten by default
Place a sentinel file at the destination with different content than the source, then run `make install-config`.
Pass: destination file is unchanged; output contains a warning mentioning the skipped file.
Fail: destination file is overwritten, or no warning printed.

## TC-003 — FORCE=1 overwrites
Run `make install-config FORCE=1` with the same sentinel file in place.
Pass: destination file now matches source.
Fail: file not overwritten.

## TC-004 — make install still works
Run `make install` end-to-end.
Pass: exits 0, `hv` binary is present at `$(go env GOPATH)/bin/hv`.
Fail: any error.
