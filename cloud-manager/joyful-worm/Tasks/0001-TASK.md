# Task 0001 — Update Makefile install-config target

## Goal
Replace the blind `cp -R` with a safe-by-default copy that warns on conflicts and requires `FORCE=1` to overwrite.

## Target repo(s)
`hive-deck-pro`

## Steps
1. Open `Makefile` in hive-deck-pro
2. Replace the `install-config` target body with logic that:
   - Iterates over every file in `$(CONFIG_SRC)` recursively
   - For each file: if it does not exist at the destination → copy it
   - For each file: if it already exists at the destination and contents differ → print a warning and skip (unless `FORCE=1`)
   - If `FORCE=1` is set → overwrite unconditionally (old behavior)
3. Update the `## install-config:` help comment to document the `FORCE=1` flag
4. Verify `make install-config` works on a clean destination
5. Verify `make install-config FORCE=1` overwrites existing files

## Definition of done
- `make install-config` on a fresh destination copies all files
- `make install-config` when destination files exist: skips conflicts, prints warnings, exits 0
- `make install-config FORCE=1` overwrites all files unconditionally
- `make install` (which calls `install-config`) still works end-to-end
