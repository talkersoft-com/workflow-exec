# Task 0002 — MCP tools for Secret Bindings

## Goal
Expose the new API as cloud-manager-mcp tools, matching the camelCase/public-id contract.

## Tools
- `cloud_secret_binding_list`, `_get`, `_create`, `_update`, `_delete`
- `cloud_blueprint_secret_binding_attach`, `_detach`, `_list`

## Steps
1. Add the tools mirroring an existing tool pair (e.g. the playbook tools) for shape/validation.
2. Inputs use public_ids (blueprintId `bp_`, secretBindingId `sb_`); outputs camelCase.
3. Build: `python3 .cicd/build-cloud-manager.py --target mcp`.
4. Note in RESULT.md that the operator must `/mcp` reload to pick up new tools.

## Definition of done
MCP builds; the tools create + list + attach the two example bindings against the live API.
Run `Test/0002-TEST.md`.
