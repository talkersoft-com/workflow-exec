# Task 0000 — Initialize Workspace
## Goal
Verify the workspace is clean and create the feature branch across all repos.
## Steps
1. `hv_status` on the deck (see `../deck.md`).
2. Clean on default branch -> `hv_init`. Clean on a feature branch with PRs merged -> `hv_next`.
3. Any repo dirty -> STOP. (Exception: dirty with a merged PR -> hv_stash -> hv_next -> hv_unstash.)
4. `hv_status` again -> confirm all on the new feature branch, all clean.
5. Record the generated branch name in `../deck.md`.
## Definition of done
All repos on the new feature branch, all clean. Branch recorded in deck.md.
