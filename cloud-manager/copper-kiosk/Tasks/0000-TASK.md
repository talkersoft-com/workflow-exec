# Task 0000 — Initialize Workspace

## Goal
Put the deck on a fresh execution branch and record it.

## Steps
1. Run `hv_status deck: "cloud-manager"`
2. All repos clean on the default branch → `hv_init`; all repos clean on a feature branch →
   `hv_next` (mints a new branch from origin/main across the deck)
3. Any repo dirty → STOP and surface. (This scaffold is on main, so it arrives tracked and clean;
   runtime edits to it — checkboxes, Retro, Results — are expected dirt that ships at the end.)
4. Run `hv_status` again — all repos on the same new feature branch
5. Record the generated branch name in `../deck.md` under "## Branch"
6. Enable the `marketplace` feature flag for the duration of this workflow
   (`cloud_feature_flag_update_enabled key:"marketplace" enabled:true`) — Phase 6 verifies the
   flag-off state explicitly before ship.

## Definition of done
All repos on the recorded execution branch; deck.md Branch no longer TBD; marketplace flag enabled.
