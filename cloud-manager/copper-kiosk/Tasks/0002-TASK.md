# Task 0002 — Marketplace storefront

## Goal
Browse published blueprints; create a VM from one in three clicks; land on the VM detail page.

## Steps
1. MarketplacePage over `GET /api/v1/marketplace`: card per published blueprint — name,
   description, base-image OS badge (from the vmi config metadata in the listing), playbook count
   ("image only" when zero); playbook names in an expandable detail (decided default).
2. "Create VM" on a card opens a MODAL (decided default): host select (reuse the wizard's host
   source), VM name input (same validation regex as the wizard), memory/diskSizeGb pre-filled
   from blueprint defaults, editable.
3. Submit → `POST /api/v1/marketplace/{blueprintId}/instantiate` → navigate to
   `/virtualmachine/{virtualMachineId}`. Surface API errors in the modal (name taken, validation).
4. SASS per the theme system; build green.

## Definition of done
Seeded `jammy` and `postgres-jammy` render as cards with correct badges/counts; instantiating
`jammy` lands on its VM detail page with the VM provisioning.
