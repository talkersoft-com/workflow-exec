# Task 0001 — Plumbing

## Goal
Everything the pages need, with zero visible change while the flag is off.

## Steps (cloud-manager-web only)
1. `src/services/api.ts`: add blueprint CRUD + lifecycle (`publish`/`archive`) + ordered playbook
   attach/reorder/detach, marketplace list, and instantiate functions — camelCase payloads,
   public ids, same error envelope handling as existing calls. Shapes per PLAN.md Background
   (instantiate: `{hostId, name, description?, memory?, diskSizeGb?}` →
   `{virtualMachineId, runPlan}`).
2. Redux: `blueprintsSlice` + `marketplaceSlice` modeled on `vmsSlice` (async thunks, status
   fields, selectors).
3. Routes `/marketplace`, `/blueprints`, `/blueprints/:id` rendering placeholder pages; nav items
   "Marketplace" (peer of Virtual Machines) and "Blueprints" adjacent — ALL gated on the
   `marketplace` feature flag using the exact `playbooks`-flag pattern in the app shell.
4. Build: `python3 .cicd/build-cloud-manager.py --target web`.

## Definition of done
Build green; flag off → app renders with no new nav/routes; flag on → nav items appear and
placeholder pages render without console errors.
