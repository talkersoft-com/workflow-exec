# Task 0001 — Remove continue_transaction; restore infinite await_merge loop

## Goal
Remove all two-phase `continue_transaction` code and replace the `await_merge` case in `ops/ship.go` with an inline infinite polling loop.

## Target repo
`hive-deck-pro` at `~/workspace/hive-deck-pro/hive-deck-pro`

## Steps

1. **`ops/contract.go`** — remove `ContinueTransaction bool` field from `ShipInput`

2. **`ops/ship.go`** — replace the `await_merge` case with:
```go
case config.PRModeAwaitMerge:
    plan, err := resolve.Build(l)
    if err != nil {
        return "", err
    }
    const pollInterval = 30 * time.Second
    time.Sleep(pollInterval)
    for {
        open := 0
        for _, repo := range plan.Repos {
            open += len(github.ListOpenPRs(repo.Dest))
        }
        if open == 0 {
            nextBranch := namegen.Generate()
            if err := checkout.Run(l, checkout.Options{
                RequireMergedPR: false,
                NextBranch:      nextBranch,
            }); err != nil {
                return "", err
            }
            return fmt.Sprintf("\nmerged: all PRs merged → transitioned to %s", nextBranch), nil
        }
        time.Sleep(pollInterval)
    }
```
   Also remove `awaitAndTransition` helper and the `ContinueTransaction` early-return block if present.

3. **`mcp/src/tools/ship.ts`** — remove `continue_transaction` parameter from schema and handler. Update await_merge hint to:
   `PRs are open and waiting to be merged. hv_ship is polling — it will transition automatically once all PRs are merged.`

4. Run `CGO_ENABLED=0 go build ./...` — must pass.

## Definition of done
`continue_transaction` is gone from all three files. `go build` passes.
