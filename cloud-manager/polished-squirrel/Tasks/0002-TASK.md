# Task 0002 ‚Äî Add open_browser config flag

## Goal
Add `open_browser: bool` to the ship config and open a browser tab for each PR URL after `ship.Run` succeeds.

## Target repo
`hive-deck-pro` at `~/workspace/hive-deck-pro/hive-deck-pro`

## Steps

1. **`internal/config/config.go`** ‚Äî add `OpenBrowser bool` to the `ShipSetup` struct:
```go
OpenBrowser bool `yaml:"open_browser"`
```

2. **`ops/ship.go`** ‚Äî after `ship.Run(...)` succeeds and before the `switch` on `PRMode`, add:
```go
if l.Setup.Ship.OpenBrowser {
    browserCmd := "open"
    if runtime.GOOS != "darwin" {
        browserCmd = "xdg-open"
    }
    openPlan, _ := resolve.Build(l)
    for _, repo := range openPlan.Repos {
        for _, pr := range github.ListOpenPRs(repo.Dest) {
            exec.Command(browserCmd, pr.URL).Start()
        }
    }
}
```
   Add imports: `"os/exec"`, `"runtime"`.

3. **`workflow-configuration/config.yaml`** ‚Äî add `open_browser: true` under `ship:`.

4. Run `CGO_ENABLED=0 go build ./...` ‚Äî must pass.

## Definition of done
`OpenBrowser` field exists in config. `go build` passes. `config.yaml` has `open_browser: true`.
