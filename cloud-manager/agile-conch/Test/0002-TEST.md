# Test 0002 — VAULT_ADDR FQDN

## TC-001 — envvars carries FQDN
`grep VAULT_ADDR /kvm-automator/playbook-runs/<recent-run>/env/envvars` matches `https://ubuntu-server.talkersoft.com:8200`.

## TC-002 — TLS validates from porch box
`curl https://ubuntu-server.talkersoft.com:8200/v1/sys/health` (NO `-k`) returns 200/2xx.

## TC-003 — TLS validates from a target VM
SSH to a target VM, run the same curl. Returns 200/2xx without `-k`.
