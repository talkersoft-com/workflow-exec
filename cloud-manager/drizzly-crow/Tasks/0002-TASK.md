# Task 0002 — Create workflow-configuration Repo + Seed + Add to Deck

## Goal

Create the `talkersoft-com/workflow-configuration` GitHub repo, seed it with current `~/.hv/` contents, and add it to the cloud-manager deck's `config` node.

## Steps

1. Create the GitHub repo:
```bash
gh repo create talkersoft-com/workflow-configuration --private --description "hive-deck configuration — decks, MCPs, modules" 2>&1
```

2. Clone it into the deck:
```bash
mkdir -p ~/workspace/cloud-manager/config
git clone https://github.com/talkersoft-com/workflow-configuration.git ~/workspace/cloud-manager/config/workflow-configuration
```

3. Seed with current `~/.hv/` contents (excluding machine-specific or secret files):
```bash
cp ~/.hv/config.yaml ~/workspace/cloud-manager/config/workflow-configuration/
cp -r ~/.hv/decks ~/workspace/cloud-manager/config/workflow-configuration/
cp ~/.hv/mcps.yaml ~/workspace/cloud-manager/config/workflow-configuration/ 2>/dev/null || true
cp ~/.hv/modules.yaml ~/workspace/cloud-manager/config/workflow-configuration/ 2>/dev/null || true
```

4. Add a `.gitignore` to the repo (exclude any local/secret overrides):
```
*.env
*.local
```

5. Commit and push the seed:
```bash
cd ~/workspace/cloud-manager/config/workflow-configuration
git add -A && git commit -m "chore: seed from ~/.hv" && git push -u origin main
```

6. Add to `~/.hv/decks/cloud-manager.yaml` config node:
```yaml
config:
  repos:
    - talkersoft-com/workflow-configuration
  symlinks:
    - ~/.hv
    - ~/.hv/toolkit/db
  show_in_workspace: true
```

7. Sync local `.hv` copy in hive-deck-pro:
```bash
cp ~/.hv/decks/cloud-manager.yaml /Users/talker/workspace/hive-deck-pro/hive-deck-pro/.hv/decks/cloud-manager.yaml
```

## Definition of done

Repo exists on GitHub with config files committed. Appears in `hv status cloud-manager` under the `config` node. `cloud-manager.yaml` updated.
