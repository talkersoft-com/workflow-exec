# Task 0003 — make configure Script + Shell Sourcing Reminder

## Goal

Add `scripts/configure-hv-home.sh` and `make configure` target to hive-deck-pro. After running `make configure`, `~/.hv/.env` contains `HV_HOME` pointing at the provisioned `workflow-configuration` repo.

## Target repo

`hive-deck-pro`

## Steps

1. Create `scripts/configure-hv-home.sh`:
```bash
#!/usr/bin/env bash
set -euo pipefail
REPO_PATH="$HOME/workspace/cloud-manager/config/workflow-configuration"
ENV_FILE="$HOME/.hv/.env"
mkdir -p "$(dirname "$ENV_FILE")"
if [ -d "$REPO_PATH" ]; then
  echo "export HV_HOME=\"$REPO_PATH\"" > "$ENV_FILE"
  echo "hv: wrote HV_HOME=$REPO_PATH to $ENV_FILE"
  # Check if shell already sources .env
  for rc in ~/.bashrc ~/.zshrc; do
    if [ -f "$rc" ] && ! grep -q "hv/.env" "$rc"; then
      echo "hv: add this line to $rc to activate HV_HOME automatically:"
      echo "    [ -f ~/.hv/.env ] && source ~/.hv/.env"
    fi
  done
else
  echo "hv: workflow-configuration not found at $REPO_PATH — skipping HV_HOME"
fi
```

2. Make it executable: `chmod +x scripts/configure-hv-home.sh`

3. Add `configure` target to `Makefile`:
```makefile
.PHONY: configure
configure:
	@bash scripts/configure-hv-home.sh
```

4. `make configure` must run without errors.

5. Verify `~/.hv/.env` was written:
```bash
cat ~/.hv/.env
```

## Definition of done

`scripts/configure-hv-home.sh` exists and is executable. `make configure` succeeds. `~/.hv/.env` contains `export HV_HOME=...`.
