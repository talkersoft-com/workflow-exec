# Task 0000 — Initialize Workspace (and verify #1.5 dependency)

## Goal
Verify the workspace is clean, create the feature branch across all repos, and confirm the #1.5
schema this workflow depends on is present.

## Steps
1. Run `hv_status` on the deck (see `../deck.md` for the exact call)
2. If all repos are clean and on the default branch → run `hv_init`
3. If all repos are clean and on a feature branch with all PRs merged → run `hv_next`
4. If any repo is dirty → STOP. Investigate before proceeding.
   - Exception: dirty repo with a merged PR → hv_stash → hv_next → hv_unstash
5. Run `hv_status` again to confirm all repos are on the new feature branch, all clean
6. Record the generated branch name in `../deck.md` (replace the `TBD` in the Branch field)
7. **Verify #1.5 landed:** confirm `marketplace.secrets` exists, `marketplace.secret_bindings` has
   a `secret_id` column, and `vm.vm_secret_bindings` exists (via db-toolkit schema tools). If any
   are missing → STOP and surface that workflow #1.5 (Secret Manager) must be merged first.

## Definition of done
All repos on the new feature branch, all clean; branch recorded in deck.md; #1.5 schema confirmed
present.
