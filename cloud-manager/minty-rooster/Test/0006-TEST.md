# Test 0006 — Deploy gate

## TC-001 — API alive
`curl -sS -o /dev/null -w "%{http_code}" http://localhost:5250/api/v1/Host`
- Pass: 200 / 401 / 403.
- Fail: 5xx, connection refused, or timeout.

## TC-002 — Web alive
Discover web port from systemd / nginx / API static config and curl `/`.
- Pass: 200 or 304.
- Fail: 5xx or no response.

## TC-003 — mcp build clean + lockfile pristine
After `npm ci && npm run build` in cloud-manager-mcp/:
- Pass: exit 0; `git status -- package-lock.json` shows no change.
- Fail: build error OR lockfile drift.

## TC-004 — No out-of-scope edits
`git diff --name-only origin/main` across cloud-manager-api,
cloud-manager-web, vorch-service, vorch-lib.
- Pass: zero files changed in those repos.
- Fail: any file changed in any of them.
