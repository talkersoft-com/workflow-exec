# Test 0005 — Coverage assertion

## TC-001 — Coverage table exists
- Pass: `cloud-manager-mcp/COVERAGE.md` exists (or the section is in README.md)
  and lists every `[Http*]`-decorated route on VirtualMachineController.
- Fail: file missing or table incomplete.

## TC-002 — Every endpoint mapped
For each endpoint row:
- Pass: a non-empty value in either the "Tool" column or the "Excluded
  (why)" column.
- Fail: any row with both columns empty.

## TC-003 — Excluded list matches plan
- Pass: only `retry-teardown` (and any non-VM endpoints out of scope)
  appear in the Excluded column.
- Fail: a different endpoint is silently excluded, or an excluded
  endpoint is missing a reason.
