# Task 0003 — Implement `cloud_vm_ip_address_update`

## Goal
Expose `PATCH /api/v1/VirtualMachine/{publicId}/ip-address` as an MCP tool.

## Steps
1. Open `cloud-manager-mcp/src/tools/vms.ts`.
2. Add `cloud_vm_ip_address_update` with input schema:
   - `publicId: string` (required, URL only, NOT in body)
   - `ipAddress: string` (required)
3. Strip `publicId` before serialising; send body `{ ipAddress }`.
4. Call: `PATCH /api/v1/VirtualMachine/${publicId}/ip-address`.
5. Return the response JSON unchanged.
6. Description string should explicitly note this tool is for operator
   overrides — vorch-service already calls this endpoint post-create
   internally.
7. Register in `cloud-manager-mcp/src/index.ts`.
8. `npm run build` — clean.

## Definition of done
- Tool registered, builds clean.
- Description string mentions "operator override" so the LLM doesn't reach
  for this tool on every VM-create flow.
