# Task 0000 — Initialize Workspace

## Goal
Verify the workspace is clean, confirm the series dependencies are merged, and create the
feature branch across all repos in the deck.

## Steps
1. Run `hv_status` on the deck (see `../deck.md` for the exact call)
2. If all repos are clean and on the default branch → run `hv_init`
3. If all repos are clean and on a feature branch with all PRs merged → run `hv_next`
4. If any repo is dirty → STOP. Investigate before proceeding.
   - Exception: dirty repo with a merged PR → hv_stash → hv_next → hv_unstash
5. Verify the series dependencies are merged to main: P2 (used-by/graph APIs), P5 (resolution
   preview), P8 (C-DECO — `<SecretValue>`, context bar, `useVarDecorations`). Check that their
   surfaces exist on origin/main (e.g. entity graph/usedBy endpoints, the C-DECO exports in
   cloud-manager-web). If any is missing → STOP and surface to operator; this terminal plan
   cannot run early.
6. Run `hv_status` again to confirm all repos are on the new feature branch, all clean
7. Record the generated branch name in `../deck.md`

## Definition of done
All repos on the new feature branch, all clean. P2/P5/P8 surfaces confirmed on main. Branch
name recorded in deck.md.
