selected renders the winner (source + level), the overridden chain, and the effective value
from the C-RES preview response. TC-002 secret masking: hovering a secret-classified variable
shows the engine's literal `***` and the `sref_…` provenance; assert the cleartext value

context, no call: with no context selected, hover renders classification + definition info
and the network layer records ZERO resolution-preview calls. TC-004 debounce + cache:
repeated hovers over different identifiers in the same context reuse one cached preview
response — no duplicate calls until doc revision or context changes. TC-005 context
persistence: choosing a context, navigating away, and returning to the same playbook restores
the choice from localStorage; a different playbook starts with its own remembered (or empty)
context. TC-006 missing amplification: a required variable the preview reports as
`missingRequired` now shows the bold gutter dot; the same variable with a context where it
resolves shows the badge only. TC-007 `--target web` build passes cleanly.
