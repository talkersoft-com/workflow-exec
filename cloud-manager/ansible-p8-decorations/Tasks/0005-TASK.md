## Steps
1. No-context behavior pass: every editor page works with no context selected — decorations
   from classification only, required badge without missing-state, hover shows definition
   info, zero resolution calls. Verify the context bar's empty state is unobtrusive.
2. Unregistered-variable behavior pass: visual silence (ordinary coloring) + the quiet
   tooltip hint only — confirm no registration UX crept in (that is P9's harvest flow, not

3. Graceful degradation without P4/P5 data: editor pages with an empty registry, a missing

4. Perf: verify the hover cache (one preview call per context per revision) and the 300ms
   decoration re-scan debounce under rapid typing; no measurable input latency regression in

   additive layer); both themes; all four masking surfaces from Task 0004 re-checked.
7. Tests: degradation cases above as component tests; an export-surface test importing
   C-DECO pieces from their public paths.
8. Build: `python3 .cicd/build-cloud-manager.py --target web` must pass.

Done when: every degradation path is silent and correct, hover/decoration perf budgets hold,
C-DECO is exported for P9, P7 editor behavior is regression-free, web build clean.
