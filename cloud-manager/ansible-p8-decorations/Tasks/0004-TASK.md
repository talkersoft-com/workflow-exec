   assignment/blueprint `VarsEditor`, P1 host/group var forms, previews and detail pages,
   `RunDetailPage` and the run vars panel; the `rsnap` (masked run snapshot) shape from P5 vs
   the raw `varsSnapshot`; P4's `$secretRef` write rules.
2. Shared `<SecretValue>` component: the single rendering path for any C-VAR-classified

   the grep pattern used and assert zero hits at completion.

8. Build: `python3 .cicd/build-cloud-manager.py --target web` must pass.

Done when: every var-value render path in cloud-manager-web goes through `<SecretValue>`,
secret writes are ref-only in the UI, run pages read the masked snapshot, and the audit grep
