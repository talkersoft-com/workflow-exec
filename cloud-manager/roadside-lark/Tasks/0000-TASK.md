# Task 0000 — Initialize Workspace

## Goal
Verify the workspace is clean and create the feature branch across all repos in the deck.

## Steps
1. Run `hv_status` on the deck (see `../deck.md`).
2. All clean on default branch -> `hv_init`. All clean on a feature branch with PRs merged -> `hv_next`.
3. Any repo dirty -> STOP and investigate. (Exception: dirty repo with a merged PR -> hv_stash -> hv_next -> hv_unstash.)
4. Run `hv_status` again to confirm all repos on the new feature branch, all clean.
5. Record the generated branch name in `../deck.md`.

## Definition of done
All repos on the new feature branch, all clean. Branch name recorded in deck.md.
