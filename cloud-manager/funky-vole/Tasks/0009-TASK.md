# Task 0009 тАФ Web: Required collections panel + Add-requirement modal + banner

## Goal
`PlaybookDetailPage` gains a "Required collections" panel listing requirements with status badges; "+ Add requirement" modal; red banner above when any requirement is unmet on the selected controller.

## Target repo(s)
`cloud-manager-web`

## Steps

1. **`api.ts`**: add helpers under a new `playbookRequirements` namespace:
   - `list(pid)` тЖТ `GET /api/v1/ansible/playbooks/{pid}/collection-requirements`
   - `create(pid, body)` тЖТ `POST /api/v1/ansible/playbooks/{pid}/collection-requirements`
   - `delete(pcreqId)` тЖТ `DELETE /api/v1/ansible/collection-requirements/{pcreqId}`

2. **New slice** `src/store/slices/playbookCollectionRequirementsSlice.ts`:
   - Entity adapter keyed by `id`.
   - Thunks: `fetchForPlaybook(pid)`, `createRequirement({ pid, body })`, `deleteRequirement(pcreqId)`.
   - Register in `store/index.ts`.

3. **`PlaybookDetailPage.tsx`** additions:
   - `useEffect` to dispatch `fetchForPlaybook(pid)` on mount.
   - Below "Arguments", new section `<section>` with title "Required collections" and a `<table>`:
     - Columns: `namespace.name`, min version, max version, status badge on selected controller, remove icon.
     - Empty state: "No collection requirements set."
     - Status computed by joining `requirement.collectionId` with the selected controller's `host_collections` (already in `hostCollectionsSlice` from lilac-redpanda).
   - "+ Add requirement" button тЖТ open `<Modal>` with:
     - Dropdown of catalog collections (from `collectionsSlice`).
     - Min version + Max version text inputs (optional).
     - "Add" submits via `createRequirement` thunk.
   - **Banner** above the panel: red if any row's status != Installed; text `"Missing required collections for the current controller."` with a "Fix" `<Link>` to `/ansible/collections`.

4. **TypeScript + build**:
   ```
   cd vm-infra/cloud-manager/cloud-manager-web
   npx tsc --noEmit
   npm run build
   ```
   Both clean.

5. Write `Retro/CHANGE-0009.md`.

## Definition of done
- New slice + API helpers + panel + modal + banner all in place.
- `tsc --noEmit` + `npm run build` clean.
- `Retro/CHANGE-0009.md` exists.
