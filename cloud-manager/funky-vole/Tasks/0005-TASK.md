# Task 0005 — Requirements CRUD (Phase B foundation)

## Goal
DTOs + Service + 3 endpoints exposing `playbook_collection_requirements`.

## Target repo(s)
`cloud-manager-api`

## Steps

1. **DTO** at `src/Models/CloudManager.DTO/Models/PlaybookCollectionRequirement.cs`:
   - Fields: `Id` (public_id), `PlaybookId`, `CollectionId`, `CollectionNamespace`, `CollectionName`, `MinVersion`, `MaxVersion`, audit columns.

2. **AutoMapper profile** — extend `AnsibleCollectionProfile.cs`:
   - `CreateMap<EntityModel.PlaybookCollectionRequirement, DtoModel.PlaybookCollectionRequirement>()` with `Id ← PublicId`, FK fields Ignore (services populate).

3. **Service interface** at `src/Services/CloudManager.Data.Services/Interface/IPlaybookCollectionRequirementService.cs`:
   ```csharp
   public interface IPlaybookCollectionRequirementService
   {
       Task<IEnumerable<DtoModel.PlaybookCollectionRequirement>> ListByPlaybookAsync(string playbookPublicId);
       Task<DtoModel.PlaybookCollectionRequirement> CreateAsync(string playbookPublicId, string collectionPublicId, string? minVersion, string? maxVersion);
       Task DeleteAsync(string pcreqPublicId);
   }
   ```

4. **Implementation** at `src/Services/CloudManager.Data.Services/PlaybookCollectionRequirementService.cs`:
   - List: load by playbook public_id, denormalize collection namespace+name.
   - Create: 409 on duplicate `(playbook_id, collection_id)`.
   - Delete: 404 if not found.

5. **Controller** `src/CloudManager.API/Controllers/PlaybookCollectionRequirementsController.cs`:
   - `GET /api/v1/ansible/playbooks/{pid}/collection-requirements` → `ListByPlaybookAsync`
   - `POST /api/v1/ansible/playbooks/{pid}/collection-requirements` → `CreateAsync` with body `{ collectionPublicId, minVersion?, maxVersion? }`
   - `DELETE /api/v1/ansible/collection-requirements/{pcreqId}` → `DeleteAsync`
   - Error envelope: known exceptions → 404/409; unknown → 500 with generic message.

6. **DI**: register in `ConfigureServices.cs`.

7. **Build**: `dotnet build CloudManager.sln -c Release` clean.

8. Write `Retro/CHANGE-0005.md`.

## Definition of done
- DTO + Mapper + Service + Controller exist.
- 3 endpoints accessible (test with `curl` post-deploy in Phase 11).
- `dotnet build` clean.
- `Retro/CHANGE-0005.md` exists.
