# Task 0008 — Web: controller selector + Collections page integration + All-controllers summary

## Goal
Operator picks the active controller via a dropdown on `AnsibleCollectionsPage` and `AnsibleCollectionDetailPage`. Selection persists in `localStorage` (`cm.selectedControllerHostId`). All status badges and actions on the page key off the selected host's `host_collections`. Detail page also shows a compact "All controllers" summary table.

## Target repo(s)
`cloud-manager-web`

## Steps

1. **New Redux slice** at `src/store/slices/selectedControllerSlice.ts`:
   - State: `{ hostId: string | null }`.
   - Action: `setSelectedController(hostId: string)`. Side-effect (in the action or via thunk): write to `localStorage`.
   - Initial state: read from `localStorage` if set.
   - Register the reducer in `store/index.ts`.

2. **Hosts type extension** at `src/types/`: add `isController: boolean` to the `Host` type.

3. **`AnsibleCollectionsPage.tsx`** rewire:
   - Replace `hosts[0]` derivation with `controllerHosts = hosts.filter(h => h.isController)`.
   - Use the Redux-stored selected id; default to `controllerHosts[0]?.id` if unset.
   - Render a `<select>` above the table titled "Controller: <select>" with options being `controllerHosts.map(h => h.name)`.
   - On change: dispatch `setSelectedController(h.id)` + re-fetch `host_collections`.

4. **`AnsibleCollectionDetailPage.tsx`** rewire:
   - Same selector pattern.
   - Add a new section below the existing single-host "Install state on …": **"All controllers"** — table with one row per controllerHost, columns `name`, `status`, `version`. Read-only.

5. **TypeScript + build**:
   ```
   cd vm-infra/cloud-manager/cloud-manager-web
   npx tsc --noEmit
   npm run build
   ```
   Both clean.

6. Write `Retro/CHANGE-0008.md`.

## Definition of done
- New slice exists, registered in store, persists to localStorage.
- Collections list page renders the dropdown.
- Detail page renders the All-controllers summary.
- `tsc --noEmit` + `npm run build` clean.
- `Retro/CHANGE-0008.md` exists.
