# Task 0000 — Verify Workspace

## Goal
Confirm all repos are on a single feature branch and clean. Record the branch
name in `deck.md` for downstream tasks and the final ship.

## Steps
1. Run `hv_status  deck: "cloud-manager"`
2. Check the reported branch:
   - All 14 repos on the **same feature branch** and clean → record that branch
     name in `../deck.md` under `## Branch`, replacing `TBD`.
   - All 14 repos on the **default branch** and clean → run
     `hv_next  deck: "cloud-manager"`, re-run `hv_status`, then record the
     new branch name in `../deck.md`.
   - Any repo dirty or branches mismatched → STOP. Investigate.

## Definition of done
All 14 repos on the same feature branch, all clean. `../deck.md` `## Branch`
section shows the branch name (no longer `TBD`).
