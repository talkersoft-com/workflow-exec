# Task 0004 — Verify round-trip, build, deploy, ship

## Goal
Prove the three new pieces work together against the live system, deploy api + mcp, write the
end-of-workflow outputs, and integrate.

## Reference
`../../../workflow-plans/cloud-manager/nostalgic-jetplane/PLAN.md` — Phase 0004.

## Steps
1. **Round-trip against the live API** (attach → update → provision → list-usage):
   - Attach a secret binding to a blueprint (`cloud_blueprint_secret_binding_attach`).
   - **Update** it with `cloud_blueprint_secret_binding_update` — change `credName` and/or
     `position`; confirm via `cloud_blueprint_secret_binding_list` the change took and untouched
     fields are unchanged (send-only-provided).
   - Provision a VM from that blueprint so #2 writes a `vm_secret_bindings` usage row.
   - **`cloud_vm_secret_binding_list`** on the new VM → confirm the injected binding appears with the
     expected `credName`, `secretBindingName`, and a resolved `resolvedVaultPath` — and **no value**.
   - Confirm 404 for an unknown VM id and `[]` for a VM with no bindings.
2. **Build:** `python3 .cicd/build-cloud-manager.py --target all` passes cleanly (api + mcp).
3. **Deploy:**
   - `python3 .cicd/deploy-cloud-manager.py --target api` → smoke with `cloud_health_check` (green).
   - `python3 .cicd/deploy-cloud-manager.py --target mcp` (rebuilds dist; does NOT restart the
     running server).
   - **No EF migration** was added — do NOT run `dotnet ef database update`.
   - A failed deploy is a stop condition → write `Retro/FIX-NNN.md`, fix, retry.
4. **Write outputs BEFORE hv_integrate:**
   - `Results/RESULT.md` — outcome, branch, PR table (filled from integrate output), phase summary,
     deploy targets + smoke HTTP codes, and — near the top, because new MCP tools were added:
     > **Operator action required:** Restart cloud-manager-mcp via `/mcp` in Claude Code to pick up the new tools.
   - `Retro/LESSONS.md` — what would have helped + any FIX files written.
5. **Ship:** `hv_integrate` with the message/title/stage from `../deck.md` (`stage: "exec"`). Report
   the PR links.

## Definition of done
Round-trip green against the live API; api + mcp build clean; api deployed and healthy; mcp dist
rebuilt; `Results/RESULT.md` + `Retro/LESSONS.md` written before integrate; `hv_integrate` opened
PRs (links reported); operator `/mcp` reload note surfaced. Test 0004 passes.
