# Task 0005 — Update Configs + Final Tests + Ship

## Goal

Update `config.yaml` on Mac and server to use `pr_mode: auto_merge`. Run full test suite. Write Results + Lessons. Ship both decks.

## Steps

1. Update `~/.hv/config.yaml` on Mac — replace `auto_merge: true` + `require_merged_pr: true` with `pr_mode: auto_merge`:
```yaml
ship:
  pr_mode: auto_merge
  delete_branch_on_merge: true
  auto_default_branch: true
```

2. Sync to local `.hv` copy:
```bash
cp ~/.hv/config.yaml /Users/talker/workspace/hive-deck-pro/hive-deck-pro/.hv/config.yaml
```
(if that file exists)

3. Run full Go test suite:
```bash
cd /Users/talker/workspace/hive-deck-pro/hive-deck-pro && CGO_ENABLED=0 go test ./...
```

4. Reinstall binary:
```bash
CGO_ENABLED=0 go install ./cmd/hv/
```

5. Smoke test JSON dispatch:
```bash
echo '{"op":"status","deck":"cloud-manager"}' | hv 2>&1 | tail -3
```

6. Update server config:
```bash
ssh todd@ubuntu-server.talkersoft.com "sed -i 's/auto_merge: true/pr_mode: auto_merge/' ~/.hv/config.yaml && sed -i '/require_merged_pr/d' ~/.hv/config.yaml"
```
Then pull + rebuild on server:
```bash
ssh todd@ubuntu-server.talkersoft.com "cd ~/workspace/hive-deck-pro/hive-deck-pro && git pull && CGO_ENABLED=0 go install ./cmd/hv/ && cd mcp && npm ci --silent && npm run build"
```

7. Write `../Results/RESULT.md`
8. Write `../Retro/LESSONS.md`

9. Ship hive-deck-pro:
```
hv_ship  deck: "hive-deck-pro"
         message: "feat: pr_mode enum — replace auto_merge + require_merged_pr booleans"
         title:   "feat: pr_mode enum — replace auto_merge + require_merged_pr booleans"
```

10. Ship cloud-manager planning:
```
hv_ship  deck: "cloud-manager"
         message: "planning: cloud-manager/dewy-bulldog"
         title:   "planning: cloud-manager/dewy-bulldog"
```

## Definition of done

All tests pass. Both configs updated. Both ships complete. Results and Lessons written before cloud-manager ship.
