# Task 0004 — Update MCP Runner + All Tool Call Sites

## Goal

Change `runHv(...args: string[])` to `runHv(payload: object)` in `runner.ts`. Update every MCP tool to pass a payload object. Rebuild MCP. End-to-end test via MCP tools.

## Target repo

`hive-deck-pro` at `/Users/talker/workspace/hive-deck-pro/hive-deck-pro/mcp/`

## Steps

1. Update `mcp/src/runner.ts` — change the `runHv` signature and implementation:
   ```typescript
   export function runHv(payload: object): RunResult {
     const result = spawnSync(HV_BIN, [], {
       input: JSON.stringify(payload),
       encoding: "utf-8",
       stdio: ["pipe", "pipe", "pipe"],
     });
     // rest of error handling unchanged
   }
   ```

2. Update every call site in `mcp/src/tools/`:

   - `sync.ts`:     → `runHv({ op: "sync", deck })`
   - `prune.ts`:    → `runHv({ op: "prune", deck })`
   - `next.ts`:     → `runHv({ op: "next", deck })`
   - `stash.ts`:    → `runHv({ op: "stash", deck })`
   - `unstash.ts`:  → `runHv({ op: "unstash", deck })`
   - `teardown.ts`: → `runHv({ op: "teardown", deck })`
   - `workflow.ts`: → `runHv({ op: "workflow", deck, name })` and `runHv({ op: "workflow", deck, list: true })`

   - `list.ts`:     → `runHv({ op: "list", deck })`

3. Run `npm run build` — must pass with no TypeScript errors.
