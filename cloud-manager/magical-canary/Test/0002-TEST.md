# Test 0002 — API controller endpoint

## TC-001 — Build + deploy clean
`python3 .cicd/build-cloud-manager.py --target api` exits 0.
`python3 .cicd/deploy-cloud-manager.py --target api` exits 0 (deploy includes its own smoke test against `/api/v1/Host`).

## TC-002 — Endpoint responds
`curl -sS -o /dev/null -w "%{http_code}\n" http://localhost:5250/api/v1/Run` returns `200`.

## TC-003 — JSON envelope shape
`curl -sS http://localhost:5250/api/v1/Run?pageSize=5 | jq 'keys | sort'` returns at least `["items","page","pageSize","total"]`.

## TC-004 — pageSize bounded
`curl -sS http://localhost:5250/api/v1/Run?pageSize=9999 | jq '.pageSize'` returns `200` (clamp held).
`curl -sS http://localhost:5250/api/v1/Run?pageSize=0   | jq '.pageSize'` returns `1`   (clamp held).

## TC-005 — Feature-flag gate intact
With the `playbooks` flag disabled, `curl -sS -o /dev/null -w "%{http_code}\n" http://localhost:5250/api/v1/Run` returns 403 / 404 / 503 per the existing gate behaviour — NOT 200. Re-enable the flag before continuing.

## TC-006 — VaultRunToken absent
`curl -sS http://localhost:5250/api/v1/Run?pageSize=50 | jq '[.items[].vaultRunToken] | map(select(. != null)) | length'` returns `0`.
