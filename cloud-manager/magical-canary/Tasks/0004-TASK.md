# Task 0004 — Web: AnsibleHistoryPage paginated grid + VM filter

## Goal
Replace the EmptyState stub on `/ansible/history` with a real paginated grid of every playbook run, ordered newest first, filterable by VM via a combobox. URL is the source of truth for `vmId` / `page` / `pageSize`; Redux mirrors them. Each row links to `/ansible/runs/:runId`.

### 1. `services/api.ts` — extend `runs`

```typescript
// Before
runs: {
  get: (runId: string) => request<...>(`/api/v1/run/${runId}`),
  ...
  cancel: (runId: string) => request<...>(`/api/v1/run/${runId}/cancel`, { method: "POST" }),
},

// After (add `list` at the top of the runs object)
runs: {
  list: (q: { vmId?: string; playbookId?: string; status?: string; page?: number; pageSize?: number }) => {
    const params = new URLSearchParams();
    if (q.vmId)       params.set("vmId", q.vmId);
    if (q.playbookId) params.set("playbookId", q.playbookId);
    if (q.status)     params.set("status", q.status);
    if (q.page)       params.set("page", String(q.page));
    if (q.pageSize)   params.set("pageSize", String(q.pageSize));
    const qs = params.toString();
    return request<{
      items: Array<{
        id: string; playbookId: string; playbookRevisionId?: string;
        status: string; startedAt: string; completedAt: string | null;
        stats: string; errorMessage: string | null;
      }>;
      page: number; pageSize: number; total: number;
    }>(`/api/v1/run${qs ? `?${qs}` : ""}`);
  },
  get: (runId: string) => request<...>(...),
  ...
}
```

### 3. `AnsibleHistoryPage.tsx` — full rewrite

Inside the existing `FeatureGate`, render:

```tsx
const Inner: React.FC = () => {
  const [params, setParams] = useSearchParams();
  const dispatch = useAppDispatch();
  const filters = useAppSelector(selectRunsFilters);
  const vmId     = params.get("vmId") ?? "";
  const page     = Number(params.get("page") ?? "1");
  const pageSize = Number(params.get("pageSize") ?? "50");

  // Mirror URL → slice on mount + on URL change
  useEffect(() => {
    dispatch(setRunsFilters({ ...filters, vmFilter: vmId || undefined, page, pageSize }));
  }, [vmId, page, pageSize]);

  const [data, setData] = useState<{ items: any[]; total: number } | null>(null);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    setLoading(true);
    api.runs.list({ vmId: vmId || undefined, page, pageSize })
      .then(d => { setData(d); setError(null); })
      .catch(e => setError(e.message))
      .finally(() => setLoading(false));
  }, [vmId, page, pageSize]);

  const setUrl = (next: { vmId?: string; page?: number }) => {
    const merged = new URLSearchParams(params);
    if (next.vmId !== undefined) {
      if (next.vmId) merged.set("vmId", next.vmId);
      else merged.delete("vmId");
      merged.delete("page");
    }
    if (next.page !== undefined) merged.set("page", String(next.page));
    setParams(merged, { replace: false });
  };

  return (
    <div className="page">
      <div className="page__header">
        <h1 className="page__title">Run history</h1>
        <p className="page__subtitle">All playbook runs · by playbook or by VM</p>
      </div>
      <RunHistoryFilters />
      <VmFilterCombobox value={vmId} onChange={(v) => setUrl({ vmId: v })} />
      {loading && <Spinner />}
      {error && <div className="page__error">{error}</div>}
      {data && (
        <>
          <RunHistoryGrid items={data.items} />
          <Pagination
            page={page}
            pageSize={pageSize}
            total={data.total}
            onPageChange={(p) => setUrl({ page: p })}
          />
        </>
      )}
    </div>
  );
};
```

Drop the `EmptyState` import.

### 4. `RunHistoryGrid` — table component

Columns: status badge, playbook name, started at, completed at / duration, click → navigate(`/ansible/runs/${item.id}`).

Status badge maps `Queued|Running|Succeeded|Failed|Cancelled` to existing status colours (look at `cloud-manager-web/src/components/StatusBadge` if it exists, otherwise inline a small switch).

### 5. `VmFilterCombobox` — combobox component

Sources options from `api.vms.list()`. Debounce filtering. Clearing the input clears the filter and removes `?vmId=` from the URL.

### 6. `Pagination` — reusable control

Shows: `‹ Prev | Page {page} of {Math.max(1, Math.ceil(total/pageSize))} | Next ›`. Disables Prev when `page === 1`. Disables Next when `page * pageSize >= total`.
