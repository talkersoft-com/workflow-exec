# Test 0009 — End-to-end verification

Each TC below is one of the ten checks from Task 0009. All must pass.

## TC-001 — Run reaches terminal Succeeded
```bash
PGPASSWORD='P@ssw0rd!' psql -h localhost -U cloudmanager -d clouddb -c \
  "SELECT status FROM vm.playbook_runs WHERE public_id = '<runId>';"
```
Pass: `status = 3`.

## TC-002 — vault_secrets_prefix populated, vault_run_token cleared at terminal
```bash
PGPASSWORD='P@ssw0rd!' psql -h localhost -U cloudmanager -d clouddb -c \
  "SELECT vault_secrets_prefix, vault_run_token IS NULL AS cleared FROM vm.playbook_runs WHERE public_id = '<runId>';"
```
Pass: prefix matches `cloudmanager/data/playbook-secrets/<runId>`, `cleared = t`.

## TC-003 — Assignment row updated
```bash
PGPASSWORD='P@ssw0rd!' psql -h localhost -U cloudmanager -d clouddb -c \
  "SELECT last_run_status, last_applied_at FROM vm.vm_playbook_assignments WHERE last_run_id = (SELECT id FROM vm.playbook_runs WHERE public_id = '<runId>');"
```
Pass: `last_run_status = 3`, `last_applied_at` within the last 5 min.

## TC-004 — Secret landed in Vault
Operator runs (with root token):
```bash
VAULT_TOKEN=<root> vault kv get cloudmanager/playbook-secrets/<runId>/postgres-test/postgres
```
Pass: returns fields `host`, `port`, `db`, `user`, `password`.

## TC-005 — Porch journal has no `hvs.*` leak
```bash
sudo journalctl -u porch --since "10 minutes ago" | grep -E 'hvs\.[A-Za-z0-9]' | head -3
```
Pass: zero matches.

## TC-006 — Vorch journal has no ansible references

sudo journalctl -u vorch_service --since "10 minutes ago" | grep -iE 'playbook|ansible|healthprobe|collection' | head -3
```
Pass: zero matches.

## TC-007 — Single consumer on playbook-runs

sudo rabbitmqctl list_consumers | grep playbook-runs | wc -l
```

## TC-008 — Feature flag healthprobe round-trip
```bash
PGPASSWORD='P@ssw0rd!' psql -h localhost -U cloudmanager -d clouddb -c \
  "SELECT key, healthy, last_probed_at FROM membership.feature_flags WHERE key = 'playbooks';"
```
Pass: `healthy = t`, `last_probed_at` within the last probe interval (≤5 min).

## TC-009 — Porch journal shows the full run lifecycle
