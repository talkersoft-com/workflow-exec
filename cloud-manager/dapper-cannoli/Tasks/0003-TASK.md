# Task 0003 — Reinit cloud-manager deck, verify settings

## Goal
Reinitialize the cloud-manager deck so hive regenerates `.claude/settings.local.json` in all repos without `Bash(*)`.

## Steps
1. Run `hv_status` — confirm all repos are clean
2. Run `hv_next` (deck is already on a feature branch) to advance to a new branch
3. Run `hv_status` again to confirm all repos are on the new branch
4. Check `.claude/settings.local.json` in one repo (e.g. `execution-workflow`) — confirm `Bash(*)` is absent

## Definition of done
All repos on a new feature branch. `.claude/settings.local.json` in at least one repo does not contain `Bash(*)`.
