# Task 0006 — Timeline endpoint

## Goal
`GET /api/v1/vm/{publicId}/events?take=100` returns the per-VM timeline ordered by `occurred_at DESC`.

## Files
- `cloud-manager-api/src/CloudManager.API/Controllers/<VirtualMachineController>.cs`
- `cloud-manager-api/src/Models/CloudManager.DTO/Models/VmEvent.cs` (DTO; mirrors entity but no internal Ids — `public_id`, `event_type`, `payload`, `actor`, `occurred_at`).
- AutoMapper profile entry for `EntityModel.VmEvent → DTO.VmEvent`.

## Steps
1. DTO + mapper.
2. Controller method:
   ```
   [HttpGet("{publicId}/events")]
   public Task<IReadOnlyList<VmEvent>> GetEvents(string publicId, [FromQuery] int take = 100)
       => _events.GetTimelineByVmPublicIdAsync(publicId, take);
   ```
3. Returns camelCase JSON (project convention). `payload` is jsonb serialized as a nested object — preserve shape, do not stringify.
4. `take` capped server-side at 500 to avoid abuse.
5. Build clean.

## Definition of done
Build clean. `GET /api/v1/vm/{publicId}/events` returns recent events for the VM, most-recent first, with public ids only and `payload` as nested JSON.
