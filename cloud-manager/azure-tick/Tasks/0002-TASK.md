# Task 0002 — API IP-write endpoint + GET surfaces

## Goal
Surface the IP on read paths (`GET /api/v1/vm/{id}`, target DTO) and add the PATCH endpoint vorch will call.

## Target repo(s)
`cloud-manager-api`

## Steps
1. Service interface — `src/Services/CloudManager.Data.Services/Interface/IVirtualMachineService.cs`:
   ```csharp
   Task<DtoModel.VirtualMachine> UpdateIpAddressAsync(string publicId, string ipAddress);
   ```
2. Service impl — `src/Services/CloudManager.Data.Services/VirtualMachineService.cs`:
   ```csharp
   public async Task<DtoModel.VirtualMachine> UpdateIpAddressAsync(string publicId, string ipAddress)
   {
       var vm = await _context.VirtualMachines.FirstOrDefaultAsync(v => v.PublicId == publicId)
           ?? throw new KeyNotFoundException($"VM '{publicId}' not found");
       vm.IpAddress = ipAddress;
       await _context.SaveChangesAsync();
       return _mapper.Map<DtoModel.VirtualMachine>(vm);
   }
   ```
3. Controller — `src/CloudManager.API/Controllers/VirtualMachineController.cs` (or equivalent):
   ```csharp
   public record UpdateIpAddressRequest(string IpAddress);

   [HttpPatch("{publicId}/ip-address")]
   public async Task<IActionResult> UpdateIpAddress(string publicId, [FromBody] UpdateIpAddressRequest request)
   {
       try
       {
           var updated = await _service.UpdateIpAddressAsync(publicId, request.IpAddress);
           return Ok(updated);
       }
       catch (KeyNotFoundException) { return NotFound(new { Message = $"VM '{publicId}' not found" }); }
       catch (Exception ex) { _logger.LogError(ex, "Error updating VM ip address {Id}", publicId); return StatusCode(500, new { Message = "Internal server error" }); }
   }
   ```
4. GET path — verify `GET /api/v1/vm/{id}` already returns the mapped DTO (it should, since AutoMapper now includes IpAddress). Verify by code reading; no change needed if mapping flows.
5. Target DTO — extend `src/Services/CloudManager.Data.Services/PlaybookRunTargetService.cs::GetByIdAsync` (around line 47-76) to JOIN VirtualMachines and surface `IpAddress`:
   - Add `public string? IpAddress { get; set; }` to `src/Models/CloudManager.DTO/Models/PlaybookRunTarget.cs`.
   - In `GetByIdAsync`, set `IpAddress = vm?.IpAddress` on the returned DTO.
6. `dotnet build` — exit 0.
