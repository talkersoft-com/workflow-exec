# Task 0005 — Backfill script for existing VMs

## Goal
One-shot Python script that iterates `vm.virtual_machines` rows where `ip_address IS NULL`, runs `virsh domifaddr <vm.name>`, parses the IPv4, and PATCHes the API.

## Target repo(s)
`cloud-manager-api`

## Steps
1. Create `cloud-manager-api/scripts/porch/backfill-vm-ip-addresses.py`:
   ```python
   #!/usr/bin/env python3
   """One-shot backfill: populate vm.virtual_machines.ip_address from libvirt for existing VMs."""
   import re
   import subprocess
   import sys
   import psycopg2
   import urllib.request
   import json

   DB = dict(host="localhost", user="cloudmanager", password="P@ssw0rd!", dbname="clouddb")
   API_URL = "http://localhost:5250"

   def domifaddr(name: str) -> str | None:
       try:
           out = subprocess.run(["sudo", "virsh", "domifaddr", name], capture_output=True, text=True, check=False)
           m = re.search(r"(\d+\.\d+\.\d+\.\d+)/\d+", out.stdout)
           return m.group(1) if m else None
       except Exception as e:
           print(f"  domifaddr({name}) failed: {e}")
           return None

   def patch_ip(public_id: str, ip: str) -> bool:
       body = json.dumps({"ipAddress": ip}).encode()
       req = urllib.request.Request(f"{API_URL}/api/v1/vm/{public_id}/ip-address", data=body, method="PATCH",
                                     headers={"Content-Type": "application/json"})
       try:
           with urllib.request.urlopen(req, timeout=5) as resp:
               return resp.status < 400
       except Exception as e:
           print(f"  PATCH {public_id} failed: {e}")
           return False

   def main():
       conn = psycopg2.connect(**DB)
       cur = conn.cursor()
       cur.execute("SELECT public_id, name FROM vm.virtual_machines WHERE ip_address IS NULL ORDER BY name")
       rows = cur.fetchall()
       print(f"Backfilling {len(rows)} VMs without ip_address...")
       ok = skipped = 0
       for pub, name in rows:
           ip = domifaddr(name)
           if not ip:
               print(f"  {name} ({pub}) — no IP from libvirt; SKIPPING")
               skipped += 1
               continue
           if patch_ip(pub, ip):
               print(f"  {name} ({pub}) → {ip} ✓")
               ok += 1
           else:
               skipped += 1
       cur.close(); conn.close()
       print(f"\nDone. {ok} updated, {skipped} skipped.")

   if __name__ == "__main__":
       sys.exit(main() or 0)
   ```
2. Make it executable: `chmod +x cloud-manager-api/scripts/porch/backfill-vm-ip-addresses.py`.
3. Do NOT run it yet — Phase 10 deploy runs it after the migration is applied.

## Definition of done
Script exists, is executable, syntactically valid Python (`python3 -m py_compile scripts/porch/backfill-vm-ip-addresses.py` exit 0).
