# Task 0008 — Redactor for subprocess streams

## Goal
Move the `redactingWriter` from `cmd/porch/main.go` to `internal/redact/` and apply it to every place subprocess output can land: log file, AMQP log publisher, and the stdlib log.

## Target repo(s)
`vorch-service`

## Steps
1. Create `internal/redact/redact.go`:
   ```go
   package redact

   import (
       "io"
       "regexp"
   )

   var hvsPattern = regexp.MustCompile(`hvs\.[A-Za-z0-9_\-]+`)

   // Scrub replaces every hvs.* token in p with hvs.<REDACTED>.
   func Scrub(p []byte) []byte {
       return hvsPattern.ReplaceAll(p, []byte("hvs.<REDACTED>"))
   }

   // ScrubString is the string variant.
   func ScrubString(s string) string {
       return hvsPattern.ReplaceAllString(s, "hvs.<REDACTED>")
   }

   type writer struct{ w io.Writer }

   // NewWriter returns an io.Writer that scrubs every Write before forwarding.
   func NewWriter(w io.Writer) io.Writer { return &writer{w: w} }

   func (r *writer) Write(p []byte) (int, error) {
       if _, err := r.w.Write(Scrub(p)); err != nil { return 0, err }
       return len(p), nil
   }
   ```
2. Add a unit test `internal/redact/redact_test.go`:
   ```go
   package redact

   import (
       "bytes"
       "strings"
       "testing"
   )

   func TestScrubReplacesToken(t *testing.T) {
       got := ScrubString("VAULT_TOKEN=hvs.ABC123_xyz next line")
       if !strings.Contains(got, "hvs.<REDACTED>") || strings.Contains(got, "hvs.ABC123_xyz") {
           t.Fatalf("expected redaction, got: %q", got)
       }
   }

   func TestWriterScrubs(t *testing.T) {
       var sink bytes.Buffer
       w := NewWriter(&sink)
       in := []byte("token=hvs.SAMPLE_TOKEN_123;")
       n, err := w.Write(in)
       if err != nil { t.Fatal(err) }
       if n != len(in) { t.Fatalf("expected %d, got %d", len(in), n) }
       if strings.Contains(sink.String(), "hvs.SAMPLE_TOKEN_123") {
           t.Fatalf("raw token leaked: %q", sink.String())
       }
       if !strings.Contains(sink.String(), "hvs.<REDACTED>") {
           t.Fatalf("expected REDACTED in: %q", sink.String())
       }
   }
   ```
3. Update `cmd/porch/main.go`:
   - Replace the inline `redactingWriter` type + `hvsPattern` + `init`'s `log.SetOutput` wiring with calls into `internal/redact`.
   - The `init` becomes:
     ```go
     log.SetOutput(redact.NewWriter(out))
     ```
4. Update `internal/ansible/playbook_run_exec.go`:
   - Import `"github.com/talkersoft-com/vorch-service/internal/redact"`.
   - At the `cmd.Stderr = lf` line, change to:
     ```go
     cmd.Stderr = redact.NewWriter(lf)
     ```
   - In the stdout scan loop (around line 280), scrub before writing to log file AND before publishing:
     ```go
     line := scanner.Text()
     scrubbed := redact.ScrubString(line)
     if lf != nil {
         _, _ = lf.WriteString(scrubbed + "\n")
     }
     outputBuf.WriteString(scrubbed)
     outputBuf.WriteByte('\n')
     // ... log_pub.publish(scrubbed) — replaces direct `line` reference
     ```
   - Audit every other place `line` is written to a sink (search for `lf.Write`, `outputBuf.Write`, `log_pub.publish`, AMQP publishes) — wrap or scrub each.
5. Build + test:
   ```
   go test ./internal/redact/
   go build ./...
   ```
