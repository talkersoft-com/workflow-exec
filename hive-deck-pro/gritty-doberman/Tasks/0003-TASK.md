# Task 0003 — branch & next

## Goal
`branch.createAll` must base feature branches on `origin/<plan.Branch>` (the deck-configured branch) instead of the git-detected `defaultBranch(dir)`. Remove the local `defaultBranch` function from `branch.go` once unused.

## Repo
`hive-deck-pro` at `/Users/talker/workspace/hive-deck-pro/hive-deck-pro`

## Steps

### 1. internal/branch/branch.go — update createAll signature
Change:
```go
func createAll(repos []resolve.RepoPlan, branchName, deck string) error {
```
To:
```go
func createAll(repos []resolve.RepoPlan, branchName, deck, deckBranch string) error {
```

Inside the loop, replace:
```go
remoteRef := "origin/" + defaultBranch(r.Dest)
```
With:
```go
remoteRef := "origin/" + deckBranch
```

If `deckBranch` is empty (deck has no `branch:` field), fall back to `defaultBranch(r.Dest)`:
```go
base := deckBranch
if base == "" {
    base = defaultBranch(r.Dest)
}
remoteRef := "origin/" + base
```

### 2. Update callers of createAll
Both `Run` and `CreateAll` call `createAll`. Both have access to `l *config.Loaded`.
Pass `l.DeckFile.Branch`:
```go
return createAll(plan.Repos, branchName, plan.Deck, l.DeckFile.Branch)
```
and
```go
return createAll(repos, branchName, plan.Deck, l.DeckFile.Branch)
```

### 3. Remove defaultBranch from branch.go if now unused
Check if `defaultBranch` is still called anywhere in the file. If not (because deckBranch fallback is inlined), delete the function. Keep it if the empty-branch fallback uses it.

### 4. go build
```bash
cd /Users/talker/workspace/hive-deck-pro/hive-deck-pro && go build ./...
```

## Definition of done
`go build ./...` passes. Feature branches are created from `origin/<deckBranch>` when set; fall back to git-detected default when not set.
