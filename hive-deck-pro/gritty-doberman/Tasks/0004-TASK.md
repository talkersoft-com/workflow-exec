# Task 0004 — ship

## Goal
`hv ship` must: (1) use the deck's configured branch as the PR base instead of the git-detected `origin/HEAD`; (2) call `EnsureRemoteBranch` before `commitsAhead` so the base ref resolves; (3) fix the pre-flight "refuse to ship from target branch" check to use the deck branch.

## Repo
`hive-deck-pro` at `/Users/talker/workspace/hive-deck-pro/hive-deck-pro`

## Steps

### 1. internal/ship/ship.go — use deck branch as PR base

Change the `repoMeta` struct field `defaultBranch string` to `targetBranch string` (rename for clarity).

In the repo collection loop, replace:
```go
repos = append(repos, repoMeta{r, branch, defaultBranch(r.Dest)})
```
With:
```go
tb := l.DeckFile.Branch
if tb == "" {
    tb = defaultBranch(r.Dest)
}
repos = append(repos, repoMeta{r, branch, tb})
```

### 2. Move EnsureRemoteBranch before commitsAhead

In Phase 2 (PR creation loop), reorder so `EnsureRemoteBranch` runs first:
```go
for _, rm := range repos {
    if rm.r.GitHubSlug == "" {
        continue
    }
    // Ensure target branch exists on remote BEFORE measuring distance against it.
    if err := git.EnsureRemoteBranch(rm.r.Dest, rm.targetBranch); err != nil {
        fmt.Fprintf(os.Stderr, "  warn    %-30s ensure remote branch %s: %v\n", rm.r.Repo, rm.targetBranch, err)
    }
    if commitsAhead(rm.r.Dest, "origin/"+rm.targetBranch) == 0 {
        continue
    }
    url, err := createPR(rm.r.Dest, rm.targetBranch, opts.Title, opts.Body)
    ...
}
```
Add import `"github.com/talkersoft/hive-deck/internal/git"`.

### 3. Fix pre-flight check
The pre-flight currently compares `rm.branch == rm.defaultBranch`. Update to use `rm.targetBranch`:
```go
if rm.branch == rm.targetBranch {
    problems = append(problems, problem{rm.r.Repo, fmt.Sprintf(
        "on target branch %q — create a feature branch before shipping", rm.targetBranch)})
    continue
}
```

### 4. Remove the now-unused local defaultBranch function from ship.go
Only remove it if no other code in ship.go calls it after the changes above.

### 5. go build
```bash
cd /Users/talker/workspace/hive-deck-pro/hive-deck-pro && go build ./...
```

## Definition of done
`go build ./...` passes. PRs open against the deck-configured branch. EnsureRemoteBranch runs before commitsAhead. Pre-flight catches shipping from the target branch.
