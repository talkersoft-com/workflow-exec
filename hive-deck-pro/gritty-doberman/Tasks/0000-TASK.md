This workflow fixes the bug that makes `hv_ship` PR against the wrong branch. Use raw git + gh commands to ship (see Task 0006 and ORCH.md). `hv_status` is safe to use for read-only inspection.

Confirm all repos are clean on `gritty-doberman`. No init or next needed — the deck is already on the execution branch.

2. Confirm all repos are on `gritty-doberman` and clean
3. If any repo is dirty — STOP and surface to operator

## Definition of done
