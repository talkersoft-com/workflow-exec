# Task 0000 — Initialize Workspace

## Goal
Fresh execution branch, recorded.

## Steps
1. `hv_status deck: "hive-deck-pro"`
2. Clean on default → `hv_init`; clean on a feature branch → `hv_next`
3. Any dirty repo → STOP (exception: this scaffold's own runtime edits in workflow-exec)
4. `hv_status` again — all repos on the same new branch
5. Record the branch in `../deck.md` under "## Branch"

## Definition of done
All repos on the recorded branch; deck.md Branch not TBD.
