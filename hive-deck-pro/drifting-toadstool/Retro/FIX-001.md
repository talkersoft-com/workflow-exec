# FIX-001 — Task 0002: decks listing order regression in golden diff

## Failure
Test 0002 TC-003 (single-layer golden diff): `hv list decks` output differed from the
pre-change capture. Two lines moved:

- pre: `cloud-manager-inactive` before `cloud-manager`; `hive-deck-pro` before `hive-deck`
- post: alphabetical by stem

## Root cause
The old `RunDecks` sorted full glob paths (which include the `.yaml` suffix), so
`cloud-manager-inactive.yaml` sorts before `cloud-manager.yaml` (`-` 0x2D < `.` 0x2E).
The new union implementation collected the trimmed stems and sorted those, flipping the
relative order of any name that is a prefix of another.

## Fix
Sort the union by filename (with `.yaml` suffix) exactly like the original path sort, and
trim the suffix only when printing. Dedupe (machine wins) still happens before the sort.
